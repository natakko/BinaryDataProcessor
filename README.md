# Binary Data Processor (Test Task)

Приложение для обработки бинарных данных с графическим интерфейсом.

## Функционал
* **Модуль 1:** Попакетное чтение данных из `.dat`, расчет средних значений по 6 каналам и экспорт в CSV.
* **Модуль 2:** Парсинг битовых полей 32-битных структур данных.
* **Интерфейс:** Выбор файлов через проводник, индикация прогресса и асинхронная обработка (интерфейс не зависает).

## Технологии
* C# / .NET 8.0
* Windows Forms (для обеспечения стабильности UI в текущей среде)
* Асинхронное программирование (Async/Await)

## Запуск
1. Склонируйте репозиторий.
2. Откройте `KonturApp.sln` в Visual Studio 2022.
3. Нажмите **F5**.
